

# Smart Trash Collection Rover

## Project Description

An IoT based autonomous waste collection rover using ESP32.

## Features

- Automatic trash collection
- Dustbin level monitoring
- Line following
- LoRa communication
- Blynk IoT dashboard
- Remote monitoring

## Hardware

- ESP32 Dev Module
- Ultrasonic Sensors
- IR Sensors
- Motors
- Motor Driver
- LoRa Module
- LCD Display

## Software

- Arduino IDE
- Blynk IoT
- GitHub

## Communication

ESP32 transmitter sends sensor data to receiver.
Receiver sends data to Blynk Cloud using WiFi.
